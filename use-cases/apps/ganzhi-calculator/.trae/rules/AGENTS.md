- **免责声明**: 以下内容为传统文化与性格取象的表达，仅供娱乐与自我观察，不构成对现实结果的保证或建议。

## 禁用规则

### 禁止使用的词汇
以下词汇在任何输出中**严禁使用**：
- 大灾
- 血光
- 必死
- 必须化解
- 不做就会
- 祖坟
- 下降头
- 破财免灾

### 低置信度时的降级替换
当置信度较低时，使用以下替换规则：

| 原词 | 替换为 |
|------|--------|
| 一定 | 更可能 |
| 必然 | 倾向于 |
| 注定 | 可能会 |

## 渲染模板

生成输出时，使用以下模板结构：

```
{disclaimer}

【骨架】日主：{day_master}｜旺衰：{strength}｜倾向：{favorable_elements}
【结构】十神侧重：{ten_gods_focus}
【画像】优势：{strengths}；注意：{risks}
【意象】{nayin_image}
【建议】{advice_list}
```

### 模板变量说明

| 变量 | 说明 |
|------|------|
| `{disclaimer}` | 免责声明文本 |
| `{day_master}` | 日主取象（如：甲木、丙火） |
| `{strength}` | 旺衰状态 |
| `{favorable_elements}` | 喜用五行倾向 |
| `{ten_gods_focus}` | 十神侧重描述 |
| `{strengths}` | 优势列表 |
| `{risks}` | 风险/注意点列表 |
| `{nayin_image}` | 纳音意象描述 |
| `{advice_list}` | 建议列表 |

## 输出规范

1. **语气控制**: 始终保持中性、克制的建议型语气
2. **避免绝对**: 不使用绝对性断言，优先使用可能性表述
3. **正向引导**: 建议应具有可操作性，避免恐吓式表达
4. **文化尊重**: 以传统文化为背景，但明确娱乐性质
5. **实用优先**: 建议应具体、可执行，避免空泛描述
